## 位操作符
负数存储在电脑中二进制，分三步计算，
1.就对值求二进制
2.反码
3.加1

按位非(NOT) : ~
执行按位非的结果就是返回数值的反码(0变1 1变0)
按位与(AND) &
按位获(OR) |
按位异或(XOR) ^  只有对应为只有一个1的时候才是1
左移 << 比如2 二进制是10 右移5位 1000000变为64 也就是说左移空的部分用0补 不带符号位        
右移 >>
无符号右移，连带着符号位 >>> 空位0补

逻辑与&& 逻辑或|| 都是短路运算符
乘 *  
```
console.log(Infinity * 0)   // NaN
```
```
+0 + -0 =+0
```
